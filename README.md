# 🛒 Ecommerce React

E-commerce app built with React, featuring cart, filters, search and dark mode.

Aplicación de comercio electrónico desarrollada con React, que incluye carrito de compras, filtros, buscador y modo oscuro.

---

##  Demo

👉 https://ecommerce-react-lj8icgkm6-maikolpls-projects.vercel.app/

## 🚀 Features

- 🛍️ Add and remove products from cart
- 🔎 Product search
- 🎯 Product filtering
- 🌙 Dark / Light mode
- 📱 Responsive design (en progreso)
- ⚡ Fast UI with React + Vite

---

## 🧠 Technologies

- React
- TypeScript
- Vite
- CSS
